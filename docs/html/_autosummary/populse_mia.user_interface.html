populse_mia.user_interface¶

User interface module, encompass the main window and its tabs definition, as well as pop ups.

Contains:
Modules:

  • data_browser

  • data_viewer

  • main_window

  • pipeline_manager

  • pop_ups

Modules

data_browser

Handle the data browser tab.

data_viewer

Handle the data viewer tab, for advanced visualisation (in 2D and 3D modes, overlay, etc.) of volumetric medical data.

main_window

Module to define main window appearance, functions and settings.

pipeline_manager

Handle the pipeline manager tab.

pop_ups

Module that defines all the pop-ups used across the Mia software.
