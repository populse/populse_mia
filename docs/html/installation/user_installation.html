Mia’s user installation¶

Since version 2.0.0, Mia can be installed directly on the host or using virtualisation. On the host, the user will not have access to the DataViewer because the one currently available by default in Mia is based on Anatomist, which should be compiled. We therefore propose to use the BrainVISA images which are available for two free and open source virtualisation technologies: Singularity and VirtualBox.

Thus, by installing a container or a virtual machine, the user has access to all the stuffs already made available by the BrainVisa developers, such as Anatomist.

In a nutshell, it is possible to install a light version of Mia on the host, without DataViewer, or to use virtualisation to install Mia, with the DataViewer access (with an additional cost for hard disk space).
